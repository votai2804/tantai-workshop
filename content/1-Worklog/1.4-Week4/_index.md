#### 1. Practice Lab 10: Hybrid DNS with Route 53 Resolver
* **Objective:** Build a hybrid DNS system integrating on-premises DNS with AWS Route 53.
* **Content Summary:** Configured Route 53 Resolver using Outbound Endpoints (sending queries to on-premises), Inbound Endpoints (receiving queries from on-premises), and Resolver Rules to route specific domain queries.
* **Skills Gained:** Hybrid DNS configuration, Route 53 Resolver management, network routing.
