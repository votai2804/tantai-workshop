### Detailed Learning, Practice, and Project Tasks:

#### 1. Study Module 01 (Cloud Concepts Overview)
* **Objective:** Understand basic cloud concepts and AWS global infrastructure.
* **Content Summary:** Study cloud computing models, economic advantages of AWS, regional infrastructure (Regions, Availability Zones), and AWS Support plans.
* **Skills Gained:** Cloud architectural mindset, AWS infrastructure understanding.

#### 2. Lab 1: Introduction to AWS Identity and Access Management (IAM)
* **Objective:** Manage permissions and secure AWS account resources.
* **Content Summary:** Create IAM Users and Groups, write IAM access policies, and test user access permissions.
* **Skills Gained:** Identity and access management (IAM), Least Privilege configuration.

#### 3. Lab 7: Introduction to Amazon DynamoDB
* **Objective:** Learn about AWS Budgets service to manage account costs.
* **Content Summary:** Set up detailed budget thresholds for each service and configure automated notifications.
* **Skills Gained:** Advanced budget management, AWS cost optimization.

#### 4. Lab 9: Introduction to Amazon API Gateway
* **Objective:** Learn about AWS Support services and support ticket management.
* **Content Summary:** Research AWS Support packages (Basic, Developer, Business, Enterprise) and practice creating/managing support requests.
* **Skills Gained:** System troubleshooting, AWS Support interactions.

#### 5. Study Module 02 (Cloud Infrastructure / Billing)
* **Objective:** Master cost calculation and cloud infrastructure optimization methods.
* **Content Summary:** Study AWS Cost Explorer, AWS Pricing Calculator, and service pricing models.
* **Skills Gained:** Cloud cost estimation, project budget planning.
