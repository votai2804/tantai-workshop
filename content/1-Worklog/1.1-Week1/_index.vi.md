### Chi tiết nội dung học tập, thực hành và dự án:

#### 1. Tạo tài khoản AWS Free Tier
* **Mục tiêu:** Nhận $100 credit đầu tiên phục vụ cho mục đích thực tập và nghiên cứu.
* **Tóm tắt nội dung:** Đăng ký tài khoản AWS mới, liên kết thẻ thanh toán quốc tế và kích hoạt tài khoản.
* **Kỹ năng đạt được:** Quản lý tài khoản AWS, thiết lập bảo mật cơ bản.

#### 2. Thực hành: Launch an instance using EC2
* **Mục tiêu:** Tạo và quản lý máy ảo (Virtual Machine) trên đám mây AWS.
* **Tóm tắt nội dung:** Khởi chạy một EC2 instance chạy hệ điều hành Linux, cấu hình Security Group bảo mật cổng truy cập và thực hiện SSH kết nối từ xa.
* **Kỹ năng đạt được:** Quản lý EC2 compute, quản trị Linux cơ bản, quản lý Key Pairs.

#### 3. Thực hành: Set up a cost budget using AWS Budgets
* **Mục tiêu:** Thiết lập cơ chế giám sát và cảnh báo chi phí để tránh phát sinh chi phí ngoài ý muốn.
* **Tóm tắt nội dung:** Tạo một ngân sách chi phí (Budget) với ngưỡng cảnh báo 80% và gửi thông báo qua email khi chi phí thực tế vượt ngưỡng.
* **Kỹ năng đạt được:** Quản lý tài chính đám mây (FinOps), thiết lập AWS Budgets.

#### 4. Thực hành: Create a web app using AWS Lambda
* **Mục tiêu:** Xây dựng một serverless web application đơn giản.
* **Tóm tắt nội dung:** Khởi tạo hàm AWS Lambda viết bằng Node.js để sinh giao diện web và thiết lập Function URL để truy cập công cộng.
* **Kỹ năng đạt được:** Lập trình serverless, cấu hình Lambda trigger.

#### 5. Làm quen các mentor và thành viên tại AWS Vietnam
* **Mục tiêu:** Hòa nhập với môi trường thực tập và làm quen với văn hóa làm việc.
* **Tóm tắt nội dung:** Giao lưu với các anh chị mentor, tìm hiểu cơ cấu tổ chức và công cụ giao tiếp nội bộ.
* **Kỹ năng đạt được:** Giao tiếp công sở, thích nghi môi trường mới.


### Kết quả đạt được:
* Tạo tài khoản thành công và nắm được các thao tác cơ bản trên AWS Management Console.
* Hoàn thành 3 bài Lab cơ bản (EC2, Budgets, Lambda).
* Lập tài khoản Budgets cảnh báo chi phí và làm quen với lập trình Serverless.
