### Detailed Learning, Practice, and Project Tasks:

#### 1. Create AWS Free Tier account
* **Objective:** Receive the initial $100 credit for internship research purposes.
* **Content Summary:** Register a new AWS account, link a payment card, and activate basic permissions.
* **Skills Gained:** AWS account management, basic security settings.

#### 2. Practice: Launch an instance using EC2
* **Objective:** Create and manage virtual machines (VMs) on the AWS cloud.
* **Content Summary:** Launch a Linux-based EC2 instance, configure Security Groups to secure ports, and connect via SSH.
* **Skills Gained:** EC2 compute management, basic Linux administration, Key Pair management.

#### 3. Practice: Set up a cost budget using AWS Budgets
* **Objective:** Establish cost monitoring and alerts to avoid unexpected cloud charges.
* **Content Summary:** Create a cost budget with an 80% threshold and configure email notifications when actual costs exceed it.
* **Skills Gained:** Cloud Financial Management (FinOps), AWS Budgets setup.

#### 4. Practice: Create a web app using AWS Lambda
* **Objective:** Build a simple serverless web application.
* **Content Summary:** Initialize an AWS Lambda function in Node.js to generate HTML content and set up a Function URL for public access.
* **Skills Gained:** Serverless development, Lambda trigger configuration.

#### 5. Meet mentors and team members at AWS Vietnam
* **Objective:** Integrate into the internship workspace and understand work culture.
* **Content Summary:** Communicate with mentors, learn about organizational structure, and internal communication tools.
* **Skills Gained:** Professional communication, team adaptability.


### Week Achievements:
* Successfully created AWS account and understood basic operations on AWS Management Console.
* Completed 3 basic Labs (EC2, Budgets, Lambda).
* Set up AWS Budgets to alert on costs and got familiar with Serverless coding.
