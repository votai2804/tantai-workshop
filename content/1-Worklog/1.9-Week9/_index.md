### Detailed Learning, Practice, and Project Tasks:
#### 1. Practice Lab 27: Manage Resources with Tags and Resource Groups
* **Objective:** Organize, categorize, and manage AWS resources at scale.
* **Content Summary:** Assign metadata tags (Key-Value) to EC2 instances and volumes, and create Tag-based Resource Groups for bulk management.
* **Skills Gained:** AWS resource management, tagging strategy design.
#### 2. Practice Lab 33: Encrypt Stored Data with AWS KMS
* **Objective:** Protect static cloud data using key encryption.
* **Content Summary:** Create a KMS Key, configure default S3 bucket encryption, log access history via CloudTrail, and query logs with Amazon Athena.
* **Skills Gained:** Key management (AWS KMS), configuring default storage encryption, CloudTrail log analysis.
#### 3. Practice Lab 44: IAM Role & Condition
* **Objective:** Enforce access control policies using conditions.
* **Content Summary:** Create EC2/RDS administrator groups. Configure IAM Roles with conditions restricting access to corporate IP addresses.
* **Skills Gained:** Advanced IAM policies (IAM Condition), Assume Role workflows.
#### 4. Participate in AWS Event on June 20
* **Objective:** Review cloud concepts and build teamwork skills under pressure.
* **Content Summary:** Engage in team contests, analyze real-world AWS scenarios, and learn risk management and team coordination.
* **Skills Gained:** AWS certificate preparation, team decision-making under pressure, risk management.
#### 5. Project (Implementation Wk 5/6): Frontend Hosting on AWS Amplify
* **Objective:** Host parent/teacher web application interface publicly on the internet.
* **Content Summary:** Link the React/Vite project from GitHub to AWS Amplify. Set up automatic continuous deployment (CI/CD Pipeline) upon commits to the `main` branch, and configure environment variables.
* **Skills Gained:** AWS Amplify Hosting deployment, GitHub CI/CD configuration, environment variable management.
