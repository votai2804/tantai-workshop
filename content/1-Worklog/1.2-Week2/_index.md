### Detailed Learning, Practice, and Project Tasks:
#### 1. Study Module 01 (Cloud Concepts Overview)
* **Objective:** Understand basic cloud computing concepts and the global infrastructure of AWS.
* **Content Summary:** Study cloud computing models, economic benefits of AWS, regional infrastructure (Regions, Availability Zones), and AWS support plans.
* **Skills Gained:** Cloud architecture thinking, understanding of AWS infrastructure.
#### 2. Lab 1: Introduction to AWS Identity and Access Management (IAM)
* **Objective:** Administer IAM permissions and secure AWS account resources.
* **Content Summary:** Create IAM Users, Groups, write IAM access policies, and test user access permissions.
* **Skills Gained:** Identity and Access Management (IAM), Least Privilege access control.
#### 3. Lab 7: Introduction to Amazon DynamoDB
* **Objective:** Learn about AWS Budgets to manage account costs.
* **Content Summary:** Set up detailed budget thresholds for each service and configure automated notifications.
* **Skills Gained:** Advanced budget management, AWS cost optimization.
#### 4. Lab 9: Introduction to Amazon API Gateway
* **Objective:** Learn about AWS Support services and how to manage support tickets.
* **Content Summary:** Study AWS support tiers (Basic, Developer, Business, Enterprise) and practice creating and managing support cases.
* **Skills Gained:** System troubleshooting, interacting with AWS Support.
#### 5. Study Module 02 (Cloud Infrastructure / Billing)
* **Objective:** Master pricing methods and cloud infrastructure cost optimization.
* **Content Summary:** Study AWS Cost Explorer, AWS Pricing Calculator, and service pricing models.
* **Skills Gained:** Calculating cloud costs, planning project budgets.
### Week Achievements:
