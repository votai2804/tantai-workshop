### Detailed Learning, Practice, and Project Tasks:
#### 1. Create AWS Free Tier Account
* **Objective:** Receive the first $100 credit for internship and research purposes.
* **Content Summary:** Register a new AWS account, link an international payment card, and activate the account.
* **Skills Gained:** AWS account management, basic security configuration.
#### 2. Practice: Launch an instance using EC2
* **Objective:** Create and manage Virtual Machines on the AWS cloud.
* **Content Summary:** Launch an EC2 instance running Linux, configure Security Group to secure access ports, and SSH remote connection.
* **Skills Gained:** EC2 compute management, basic Linux administration, Key Pairs management.
#### 3. Practice: Set up a cost budget using AWS Budgets
* **Objective:** Set up monitoring and cost alert mechanisms to avoid unexpected costs.
* **Content Summary:** Create a cost budget with an 80% alert threshold and send email notifications when actual costs exceed the threshold.
* **Skills Gained:** Cloud financial management (FinOps), AWS Budgets configuration.
#### 4. Practice: Create a web app using AWS Lambda
* **Objective:** Build a simple serverless web application.
* **Content Summary:** Initialize an AWS Lambda function written in Node.js to generate web interfaces and set up a public Function URL.
* **Skills Gained:** Serverless programming, Lambda trigger configuration.
#### 5. Get familiar with Mentors and team members at AWS Vietnam
* **Objective:** Integrate with the internship environment and get familiar with the work culture.
* **Content Summary:** Interact with mentors, learn about the organizational structure, and internal communication tools.
* **Skills Gained:** Workplace communication, adapting to a new environment.
### Week Achievements:
* Successfully created AWS account and understood basic operations on AWS Management Console.
* Successfully completed 3 basic Labs (EC2, Budgets, Lambda).
* Set up AWS Budgets to alert on costs and got familiar with Serverless coding.
